APA ITU XAMPP

Software web server apache yang di dalamnya sudah tersedia database server mySQL dan support PHP Programming Kenapa Namanya XAMPP? (Exe) = Cross Platform, adalah kode penanda program (software) yang dapat berjalan di banyak system operasi, XAMPP itu bisa di jalankan di banyak OS (operating system) seperti windows, mac OS, dan Linux. A = Apache, adalah aplikasi web server yang bersifat gratis dan bisa dikembangkan oleh banyak orang (open source). Fungsi Apache pada XAMPP yaitu untuk menghasilkan halaman web yang benar bagi user berdasarkan kode PHP yang ditulis oleh pembuat web. M = MySQL / MariaDB, merupakan aplikasi database server yang dikembangkan oleh orang yang sama. MySQL berperan dalam mengolah, mengedit, dan menghapus daftar melalui database. P = PHP, huruf inisial untuk menunjukkan eksistensi bahasa pemrograman PHP. P = Perl (People of relational labor), bahasa pemrograman yang di gunakan untuk segala keperluan. Bahasa pemrograman ini sudah berkurang karena munculnya Bahasa-Bahasa pemrograman baru dan tren teknologi yang berubah Apache : Di dalam XAMPP, Apache digunakan untuk menjalankan dan mengelola situs web atau aplikasi web lokal Anda. MySQL : MySQL adalah sistem manajemen basis data relasional (RDBMS) yang digunakan untuk menyimpan dan mengelola data dalam bentuk tabel yang terstruktur. Dalam XAMPP, MySQL digunakan sebagai basis data untuk aplikasi web. Kalian dapat membuat, mengelola, dan mengakses basis data melalui antarmuka komando atau alat manajemen basis data visual. FileZilla: FileZilla adalah perangkat lunak FTP (File Transfer Protocol) yang memungkinkan Anda untuk mentransfer file antara komputer Anda dan server melalui jaringan. Dalam XAMPP, FileZilla digunakan sebagai alat untuk mengelola file yang terkait dengan situs web atau aplikasi web. Kalian dapat mengunggah, mengunduh, dan mengelola file di server lokal Anda. Mercury: Mercury adalah server mail yang dapat digunakan untuk mengirim dan menerima email dalam pengaturan pengembangan lokal. Ini mencakup SMTP (Simple Mail Transfer Protocol) dan POP3 (Post Office Protocol 3). Tomcat: Server aplikasi web yang mengimplementasikan spesifikasi Java Servlet dan JavaServer Pages (JSP). Dalam XAMPP, Tomcat dapat digunakan untuk mengembangkan dan menguji aplikasi web berbasis Java EE (EnterpriseÂ Edition).
